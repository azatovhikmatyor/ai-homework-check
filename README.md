# homework-check
AI powered homework checker

# Google sheet requirements
Add your sheet's url with "SHEET_URL" keyword and credential keys json file path with "KEY_FILE_PATH" keyword to .env file.
Each student must have unique ID.
Each column which shows students' grade must be named like this: CLASS1, CLASS2, CLASS3....

# To use AI
Add Api_key of Gemini with "Gemini_KEY" keyword to .env file

# Repo appearence
Repo must be like this: "https://github.com/azatovhikmatyor/python-homeworks" 
not like this: "https://github.com/azatovhikmatyor/python-homeworks.git"

# Homework structure
When homework is uploaded to the repo, the structure of folders must be like this:
lesson-1/homework/...
lesson-2/homework/...
lesson-3/homework/...
...
